# thermal-controller-iot
Система анализа и логирования температуры и влажности с Web-интерфейсом

## Основные (предпологаемые) функции
- Сбор данных с датчика BME280 каждые 30 секунд
- Анализ трендов температуры и влажности
- Система алертов при выходе за заданные диапазоны
- Веб-интерфейс с графиками в реальном времени
- Уведомления через Telegram/Email (опционально)
- Хранение данных в SQLite с ротацией 90 дней

## Архитектура системы
### Аппаратная часть:
- Raspberry Pi (3/4/Zero)
- Датчик температуры и влажности BME280

### Программная часть
- Язык: Python 3.9+
- Веб-фреймворк: Flask + REST API
- Фронтенд: Bootstrap 5 + Chart.js + jQuery
- База данных: SQLite с SQLAlchemy ORM
- Аналитика: pandas + scipy + scikit-learn
- Драйверы: RPi.GPIO, adafruit-circuitpython-dht
- Конфигурация: YAML + dotenv
- Логирование: встроенный logging с ротацией

## Старт проекта
1. Установка: sudo ./install.sh
2. Настройка: sudo ./configure.sh
3. Запуск: sudo systemctl start temperature-monitor
4. Доступ: http://raspberry-pi:8080
